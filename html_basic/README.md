# ALU Web Development - HTML Basics

This repository contains the foundational HTML project introducing document structures, essential semantic markup tags, and validation compliance.

## Learning Objectives
* Understand what HTML and the DOM are.
* Structure a valid page with a DOCTYPE, head, and body.
* Apply basic text formatting tags (`<h1>`, `<h2>`, `<h3>`, `<p>`).
* Build functional anchors using clickable images (`<a>` and `<img>`).

## Files
* `base_index.html`: The structural skeletal baseline template.
* `index.html`: Fully populated semantic webpage passing W3C validation.
