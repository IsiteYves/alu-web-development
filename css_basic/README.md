# ALU Web Development - CSS Basics

This project introduces the fundamentals of CSS (Cascading Style Sheets), detailing how to structure, style, and build responsive layouts using CSS Flexbox without external libraries.

## Learning Objectives
* Understand what CSS is and how it targets elements.
* Apply external styles using `base.css` and `styles.css`.
* Compute CSS Specificity values.
* Implement structural layouts using the CSS Flexbox model.
* Build responsive webpages that scale down gracefully on mobile viewports.

## Files
* `index.html` - Main landing page.
* `tweets.html` - Supplementary layout page.
* `base.css` - Structural core resets.
* `styles.css` - Custom structural Flexbox layout and styling adjustments.
